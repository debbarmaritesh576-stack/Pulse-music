# 🎵 Pulse Music

Beautiful offline music player for Android. No ads. No internet. Just music.

## ✨ Features
- 🎵 All formats (MP3, FLAC, WAV, AAC, OGG, OPUS)
- 🎨 Material 3 Design
- 🎚️ 10-Band Equalizer
- 📝 Lyrics Support
- 🔒 100% Offline & Private

## 🏗️ Tech Stack
- Kotlin + Jetpack Compose
- Media3 ExoPlayer
- Room Database
- Hilt DI
- Clean Architecture
