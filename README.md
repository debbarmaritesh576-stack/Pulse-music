# 🎵 Pulse Music

Beautiful offline music player for Android. No ads. No internet. Just music.

## ✨ Features

- 🎵 **All Formats** — MP3, FLAC, WAV, AAC, OGG, M4A, OPUS
- 🎨 **Material 3** — Dynamic colors, dark mode
- 🎚️ **10-Band EQ** — 8 presets + Bass Boost + Virtualizer
- 📝 **Lyrics** — Embedded + LRC support
- 📂 **Browse** — Songs, Albums, Artists, Folders, Genres
- 📋 **Playlists** — Create, edit, import M3U
- 🔍 **Search** — Instant search
- ⏱️ **Sleep Timer** — Auto-stop
- 📱 **Widget** — Home screen controls
- 🔒 **100% Offline & Private**

## 🏗️ Tech Stack

| Component | Technology |
|:---|:---|
| UI | Jetpack Compose + Material 3 |
| Audio | Media3 ExoPlayer |
| Database | Room |
| DI | Hilt |
| Architecture | MVVM + Clean Architecture |
