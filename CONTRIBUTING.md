# Contributing to Pulse Music

## How to Contribute
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Code Style
- Kotlin + Jetpack Compose
- MVVM Architecture
- Hilt for DI
- Room for database

## Building
```bash
git clone https://github.com/pulsemusic/PulseMusic.git
./gradlew assembleDebug
